# Гибкие методологии в управлении качеством образования

Одностраничный лендинг образовательного интенсива о применении гибких методологий, бережливых технологий и инструментов управления качеством в образовательных программах.

## О проекте

Проект представляет собой статическую HTML-страницу на русском языке. Лендинг описывает программу объёмом **36 академических часов** в смешанном формате: **12 часов очно** и **24 часа асинхронно**.

На странице представлены:

- ключевые параметры программы;
- целевая аудитория;
- ценность и ожидаемые результаты обучения;
- структура интенсива;
- перечень изучаемых тем;
- информация о сертификате участника.

## Технологии

- HTML5;
- Tailwind CSS через CDN;
- Fontsource Inter через CDN;
- встроенные CSS-анимации и стили;
- нативный JavaScript для появления элементов при прокрутке.

## Структура проекта

```text
.
├── index.html   # Основная страница лендинга
└── README.md    # Описание проекта
```

## Быстрый запуск

Откройте файл `index.html` в браузере любым удобным способом.

Например, можно запустить локальный статический сервер:

```bash
python3 -m http.server 8000
```

После запуска откройте страницу:

```text
http://localhost:8000
```

## Особенности интерфейса

- светлая визуальная тема с фиолетовыми акцентами;
- адаптивная вёрстка для разных размеров экрана;
- карточки с эффектом стекла;
- анимированные заголовки, свечения и фоновые элементы;
- плавное появление блоков при прокрутке.

## Назначение

Лендинг можно использовать как презентационную страницу для образовательного интенсива, курса повышения квалификации или информационного блока на сайте образовательной организации.
